Project Overview

This repository aggregates open-access materials related to the Repetition Experiments as Probes on Linguistics Analysis and Integration (REPLAI) research program, funded through PRIN 2022 MUR NextGeneration EU.

Data Sources

  • GitHub (experimental code repositories)
  • Open Science Framework (behavioral datasets, preprints)
  • University of Padova institutional repository (dissertations)
  • Conference proceedings (peer-reviewed abstracts)

Program Structure

Research Integration

All indexed materials constitute components of a unified research program:

REPLAI – PRIN 2022 MUR NextGeneration EU

Organizational principles:

  • Open-source experimental code (GitHub infrastructure)
  • Born-open behavioral data (OSF archival)
  • Collaborative stimulus development (Calignano repository)
  • Graduate-level empirical investigation (Master's dissertations)
  • Transparent dissemination (preprints, open conference proceedings)

Scope note: No independent repositories with Calignano co-authorship associated with the Maze paradigm exist external to this integrated pipeline.
